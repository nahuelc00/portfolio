Acerca de mi

Experiencia

Durante estos años trabajé dentro de las areas de atención al publico, transportista y operario. Mi última experiencia fue en una empresa de IT desarrollando el rol de developer fullstack, usando React en el front y NestJs en el back.

Arquitectura backend

Creo las API´s REST utilizando el patrón de arquitectura MVC (Model/View/Controller). Utilizo Postman para diseñar los endpoints. NestJs o Node.Js con Express son las tecnologías que uso, dando la capacidad de alojar la data en una base de datos SQL como NoSQL. Testeo las API con Jest, guardando la data de la ejecución en una base de datos local.

Desarrollo Frontend

Desarrollo de Single Pages App utilizando HTML, CSS y vanilla Javascript, Typescript o React. Para el diseño me enfoco en el paradigma mobile-first, dándole prioridad al diseño en dispositivos móviles. Utilizando programación reactiva por medio de un estado y routers para mantener actualizados los entornos y redirigir la navegación. Testeo las SPA con Cypress.
